1 - Caminho do repositorio no github
https://github.com/jmbes/librarium.git

2 - Configuracao do git local
git config --global user.name "Juliano Barbosa"
git config --global user.email "jmbes_btu@hotmail.com"

3 - Inicializacao do repositorio local
git init 
git add .
git commit -m "comentario"

4 - Define a branch do repositorio
git branch -M main

5 - Define a UTL do repositorio no github
git remote add origin https://github.com/jmbes/librarium.git

6 - Envia o conteudo para o repositorio do github
git push -u origin main
